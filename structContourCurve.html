void ContourCurve::drawFront ( pen  corePen = outlineContour,
light  light = currentlight,
real  arrowPosition = -1 
)
inline

draws the curve as if it is over others lines. This method works as following:

Firstly it draws cover-paths as curve defines, but wider than corePen's width. The color of cover-paths is same background color of currentpicture. Each segment of curve is maped to a subpath (=cover-paths), which is matched the correspondent offset. Only these subpaths are drawn.

Then it draws an other path as curve defines, with the given pen corePen. This creates a optical illusition, as if this curve is drawn over other curve.

Parameters
corePenpen to draw this curve
lightis used to determine background color for this line, which cover other lines
arrowPositionis used as the same way in draw
